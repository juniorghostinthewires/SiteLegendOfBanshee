Como jogar Legend of Banshee?

Para jogar, Baixe o JOGO e para Começar a Aventura, selecione a opção 'Novo Jogo'.
Utilize as 'Setas' do teclado ou 'W,A,S,D' para movimentar o personagem.
Utilize o 'Espaço' para executar uma ação
Utilize a tecla 'Z' para Selecionar um Personagem

(Sem Misericórdia, se o jogador perder... voltamos ao inicio!) Instruções desde o ínicio: Baixe o jogo, execute o arquivo baixado na pasta com nome do jogo clique em game.exe. Utilize ALT + G para colocar o jogo em tela cheia no seu computador.
